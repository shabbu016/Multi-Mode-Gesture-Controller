# Multi-Mode Gesture Controller

A computer vision-based system to control volume, brightness, media playback, and mouse using hand gestures.

## Features
- Volume Control
- Brightness Control
- Media Control (Play/Pause, Next, Previous)
- Mouse Control
- Mode Switching via gestures

## Tech Stack
- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- Pycaw

## Run
pip install -r requirements.txt
python volume_controller.py
